
## NestJs - Central Server (prototipo)

Server prototipo en NestJs, para Tesis de la Universidad Maimonides.
Este server simula ser un servidor central donde diferentes apps se comunican a el a para buscar contenidos y actualizacion de configuraciones.


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests    

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
